# HW01 — Rarjpeg Detector

Написать программу, определяющую, является ли заданный файл т.н. **Rarjpeg-ом** (изображением, в конец которого дописан архив) либо обычным архивом, и выводящую список файлов в архиве, если заданный файл таковым является.

Для простоты подразумевается формат архива **zip**, а не rar. Сторонние библиотеки не использовать.

**Цель:** получить навык работы с бинарными файлами, описываемыми структурами языка C.

## Критерии успеха

- Создано консольное приложение, принимающее аргументом командной строки входной файл.
- Приложение корректно определяет файл, не содержащий внутри архив (файлы для проверки прилагаются).
- Приложение корректно определяет файл, содержащий внутри архив, и выводит список файлов внутри архива (файлы для проверки прилагаются).
- Код компилируется без предупреждений с ключами компилятора `-Wall -Wextra -Wpedantic -std=c11`.
- Далее успешность определяется ревью кода.

## Справка по форматам

### JPEG

- Начало файла (SOI): `FF D8`
- Конец файла (EOI): `FF D9`

### ZIP — общая структура

```
[ Local File Header #1 + данные ][ Local File Header #2 + данные ]...
[ Central Directory Entry #1 ][ Central Directory Entry #2 ]...
[ EOCD ]
```

### EOCD (End of Central Directory) — 22 байта

| Смещение | Размер | Описание |
|----------|--------|----------|
| 0–3 | 4 | Сигнатура `50 4B 05 06` |
| 4–5 | 2 | Номер диска (легаси) |
| 6–7 | 2 | Диск с началом CD (легаси) |
| 8–9 | 2 | Кол-во записей CD на этом диске |
| 10–11 | 2 | Общее кол-во записей CD |
| 12–15 | 4 | Размер Central Directory |
| 16–19 | 4 | Смещение CD от начала архива |
| 20–21 | 2 | Длина комментария |

### Central Directory Entry — 46 байт + переменная часть

| Смещение | Размер | Описание |
|----------|--------|----------|
| 0–3 | 4 | Сигнатура `50 4B 01 02` |
| 4–5 | 2 | Версия, которой создан архив |
| 6–7 | 2 | Версия, нужная для распаковки |
| 8–9 | 2 | Битовые флаги (шифрование и т.д.) |
| 10–11 | 2 | Метод сжатия (0 = без сжатия, 8 = deflate) |
| 12–13 | 2 | Время модификации файла |
| 14–15 | 2 | Дата модификации файла |
| 16–19 | 4 | CRC-32 (контрольная сумма) |
| 20–23 | 4 | Сжатый размер файла |
| 24–27 | 4 | Несжатый размер файла |
| 28–29 | 2 | Длина имени файла (N) |
| 30–31 | 2 | Длина extra field (E) |
| 32–33 | 2 | Длина комментария к файлу (C) |
| 34–35 | 2 | Номер диска (легаси) |
| 36–37 | 2 | Внутренние атрибуты файла |
| 38–41 | 4 | Внешние атрибуты файла (права доступа) |
| 42–45 | 4 | Смещение Local File Header от начала архива |
| 46 | N | Имя файла |
| 46+N | E | Extra field |
| 46+N+E | C | Комментарий к файлу |

### Магические байты

| Формат | Байты |
|--------|-------|
| JPEG начало | `FF D8` |
| JPEG конец | `FF D9` |
| ZIP Local File Header | `50 4B 03 04` |
| ZIP Central Directory | `50 4B 01 02` |
| ZIP EOCD | `50 4B 05 06` |
