# Домашнее задание 5. Файлы и директории в GTK

Написать приложение с использованием **GTK**, отображающее файлы и директории в текущем каталоге.

Сторонние библиотеки (кроме **GLib/GTK**) не использовать.

## Требования

1. Создано приложение, при запуске рекурсивно отображающее в **GtkTreeView**-виджете файлы и директории, расположенные в директории, откуда оно было запущено.
2. Код компилируется без предупреждений с ключами компилятора `-Wall -Wextra -Wpedantic -std=c11`. Предупреждения об устаревших API допускаются.
3. Далее успешность определяется ревью кода.

## Сборка (Meson)

Нужны установленные **GTK 4** и **pkg-config** (например, `brew install gtk4 pkg-config`).

```bash
cd HW05
meson setup build
meson compile -C build
```

Исполняемые файлы: `build/hello-gtk4`, `build/gtk4_example`.
